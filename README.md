This repository contains basic Python programs that demonstrate fundamental programming concepts and problem-solving techniques. These programs are commonly used for learning Python, coding interviews, laboratory exercises, and technical assessments.

Programs Included:

1. Fibonacci Series

Generates and prints the first N numbers in the Fibonacci sequence using iterative logic.

2. Palindrome Checker

Checks whether a given string or number reads the same forward and backward.

3. Factorial Calculation

Computes the factorial of a number using both iterative and recursive approaches.

4. Prime Number Operations

Determines whether a number is prime and prints all prime numbers between 1 and 100.

5. Armstrong Number Checker

Verifies whether a given number is an Armstrong number by comparing it with the sum of its digits raised to the power of the total number of digits.

6. Number Reversal and Digit Sum

Reverses a given number and calculates the sum of its digits.

7. Second Largest Number Finder

Finds the second largest element from a list of numbers.

8. Vowel and Consonant Counter

Counts the number of vowels and consonants present in a given string.

9. Missing Number Finder

Identifies the missing number from a sequence containing N−1 numbers from the range 1 to N.

10. Anagram Checker

Checks whether two strings are anagrams by comparing their character arrangements.
